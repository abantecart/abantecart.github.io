General tips to keep in mind

There are many different things that could cause a problem with your AbanteCart installation.

Mod_security

hosting related
Mod_security is an apache module that helps to protect your website from various attacks. mod_security has certain rules for the words "pattern" used in requested URL's. If any pattern match with the requested URL under your site mod_security false positive and deny the access to it. Many people are experiencing problems because of this. Problems include "403", "500" or access denied errors, login problems, and similar. Ask your hosting support to disable individual mod_security rules or try to disable it
Try adding this in a .htaccess file in the /admin/ folder 

	
SecFilterInheritance Off

MySQL

database
MySQL database version 5.0+ or above, MySQLi.

cURL support

PHP extension
PHP CURL is required for operation of several payment gateways and systems (such as PayPal, Authorize.NET etc) and real-time shipping services (FedEx, UPS).

GD

library
GD php library is installed

ZIP

extension
ZipArchive extension is enabled

AbanteCart will not work properly (or at all) without these technologies.
