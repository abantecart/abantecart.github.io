Debug Default Template

Built-in debugger lets you easily debug any page of your storefront or admin panel.

  • Activate template Debug Mode

    Access the debugger

  • Browser code inspector

    Find proper CSS rule to edit

Activate template Debug Mode

To access the debugger, go to the Administration panel > System > Settings > System and set Template debug to On. This will give you a links for storefront and admin with the debug parameter in the URL.


 The debug label icon will appear in the top corner of the each block. After click you will find detailed information about the block.

Note

Important to backup before changes.

Browser code inspector

browser related
Almost all browsers have the Page or Code Inspector to examine the HTML and CSS of a page.

 When you select some element on your page the Rules view lists show all the CSS rules that apply to the selected element. This can help you understand where CSS file located and what rule your need to edit.
