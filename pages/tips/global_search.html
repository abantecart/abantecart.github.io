Quick Search

Search anything in admin

Quick search is tool to help you find quickly almost everything. Type text and AbanteCart search engine will give your results within the orders, text translations (language definitions), customers, settings, products etc.

The best way to deal with changing language text is do this via control panel tools.
You can use global search (admin) to find text that you are looking for edit, click on the link in the result and edit it.

After you save the result, refresh the page where to expect text to show.

Edit

When you click to edit search result item for some items popup will be open for other new page.
