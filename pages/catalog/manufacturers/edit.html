Manufacturer Name: The manufacturer name as it will appear in the front-end.

Also you can fill/edit/select the following fields:

Stores: Select the store where Manufacturer will appear on the storefront

SEO Keyword: The Search Engine Friendly URL Identifier is the name used of the manufacturer used in the URL for the manufacturer when Use SEO URL's enabled.

After you have made your changes, select the SAVE button.

Then page will reload and you will be able to add Image or any Media using Resource Library and modify custom Layout for this manufacturer by click on Layout Tab.

Layout: Select a layout override to change the look and feel of this manufacturer page. Read more about Layout in this chapter
