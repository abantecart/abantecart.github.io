Media Tab

Product's media resources

Uploading media files can be accomplished through the AbanteCart interface. No FTP or cPanel access needed. In most cases you need to upload images media resources but you also able to upload media like Audio, Video, Pdf, Archive or even HTML resources.


Add mages to product

To add additional images files go to the 'Media' tab and drag-n-drop your images or click on "Plus" button. The Resource library popup window will appear.

In the Resource Library you able to fully manage product's images: upload new images, select exiting in library, change sort order or remove images.


In order to set an main image for your product click on any uploaded image and in resource library window click on first Tab in the popup, all images mapped to the current product will appear, change sort order number of image and click on Save Sort Order button.


Maximum size of an uploaded media files depends on your server configuration and Resource Library Upload Max File Size setting in System > Settings > System.

Image file size and dimensions size

Your server may be limited for the upload file size and image's dimensions size (width/height) processing. Both related to your server configuration. We strongly recommend you contact your hosting provider and work with them to resolve the server configuration issues.


Add video to product

We recommend not uploading videos directly through AbanteCart. Instead, upload your videos through YouTube video streaming service. See how to Add YouTube video as HTML resource to product


Add pdf to product

Best way to add Downloadable PDF (Product Manual) on product's page is create new download with Activated > Before Order setting in the Files Tab. See how to Create downloads
