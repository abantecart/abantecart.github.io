Media Tab

Product's media resources

Uploading media files can be accomplished through the AbanteCart interface. No FTP or cPanel access needed. In most cases you need to upload images media resources but you also able to upload media like Audio, Video, Pdf, Archive or even HTML resources.

Add mages to product

To add additional images files go to the 'Media' tab and click on "Plus" button. The Resource library popup window will appear.

In order to set an main image for your product simply click on the "Plus image" and a popup will appear, click on first Tab in the popup, images mapped to current product will appear, change sort order of image and click on Save Sort Order button.

Maximum size of an uploaded media files depends on your server configuration and Resource Library Upload Max File Size setting in System > Settings > System.

Add video to product

We recommend not uploading videos directly through AbanteCart. Instead, upload your videos through YouTube video streaming service. See how to Add YouTube video as resource to product
