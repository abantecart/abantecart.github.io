Custom Layout

In spite of general layouts you can create individual layout for pages

You able to create custom layout for individual products, categories, manufacturers, content pages. On every product (or others) edit page there is Layout Tab. Once you modify it and save the New Individual layout will appear in Design > Layouts section.
