View Install/Upgrade History

System > Logs > Install/Upgrade History

Every time administrator installs new extension, upgrades them or does application update, create backup, there is a record created. Update history record indicates date and time of installation, with the name and user name that actually made installation. 
Version and license keys are also indicated.
