View AbanteCart Errors log

System > Logs > Errors log

A full log of every error message your store experiences. Review the error messages in case of any problems.

Or Click Clear Log button to reset error log.

Also You can disable error loging in System > Setting > System Tab. Find Log Errors and select OFF

Note

In case you can not access admin you can locate error log directly in the file /system/logs/error.txt
