Edit Weight Class

System > Localization > Weight Classes

*Weight Title: Name of the measure.

*Weight Unit: Abbreviation for the measure.

Value: Set to 1.00000 if this is your base weight.

Weight class values for conversion are defined in relation to base weight
Example:
base weight class kg set value to 1.00000
now in relation to base weight (kg) set values for other weight classes
ounce (1 Kilogram = 35.274 Ounce) set value to 35.2740
gram (1 Kilogram = 1000 Gram ) set value to 1000.0000

After you have made your changes, click save button
