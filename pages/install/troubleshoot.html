  • Warning: Your server have opcache php module enabled. Please disable it before installation!

    Solution: You can try one of the following:
    1. If PHP runs as Apache module, use an .htaccess file:
    php_flag opcache.enable Off

    2.If PHP runs as CGI/FastCGI, use a .user.ini file:
    opcache.enable=0
