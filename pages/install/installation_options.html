Installation

You can choose different AbanteCart installation options, depending on your need and technical experience. Installation can be done with simple click to more advanced options.

Click Installation

One-click install solution for AbanteCart. Download installers and virtual machines

Cpanel Installation

cPanel is a Linux based web hosting control panel that provides a graphical interface and automation tools designed to simplify the process of hosting a web site.

Cloud Installation

Cloud Hosting. Launch AbanteCart app in the cloud with one click and have them automatically backed up and monitored.

Localhost installation

You do not need to get hosting account to test run or develop with AbanteCart application. Get convenience of accessing web site and code locally.

Regular Installation

Use this approach if you do not have Cpanel on your server. The instructions are about AbanteCart Manual installation on a server.
