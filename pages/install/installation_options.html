Installation

You can choose different AbanteCart installation options, depending on your need and technical experience. Installation can be done with simple click to more advanced options.

Click Install

Create a personal trial store. If you decide to make your personal site live, you can do it and any time, with no hassle migration to hosted account.

Cpanel Installation

cPanel is a Linux based web hosting control panel that provides a graphical interface and automation tools designed to simplify the process of hosting a web site.

Cloud Installation

Bitnami Cloud Hosting. Launch AbanteCart app in the cloud with one click and have them automatically backed up and monitored.

Localhost installation

If you want to test or develop with AbanteCart, you can install it on your own computer.

Regular Installation

The instructions are about AbanteCart Manual installation on a hosting server accessible on the Internet.

Custom Install

AbanteCart can be installed with specific configuration for higher security and better performance. This is usually done for larger sites. Technical knowledge is required.
