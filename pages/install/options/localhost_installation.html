Installation on local computer

You do not need to get hosting account to test run or develop with AbanteCart application. If you want to test or develop with AbanteCart, you can install it on your own computer and get convenience of accessing web site and code locally. There are 3 most popular third party applications available that provide virtual layer with web server and database functionality. All are free. Bitnami the simpliest way and support few OS, XAMPP is available for MAC, WIN and Linux.

Bitnami Stack

Bitnami AbanteCart Stack provides a one-click install solution for AbanteCart.

XAMPP

XAMPP is a completely free, easy to install Apache distribution containing MySQL, PHP, and Perl.

Denwer

Denwer is more for russian speaking developers and community.

Bitnami AbanteCart Stack

Bitnami AbanteCart Stack
https://bitnami.com/stack/abantecart

Bitnami AbanteCart Stack provides a one-click install solution for AbanteCart. Bitnami native installers automate the setup of a Bitnami AbanteCart application stack on Windows, OS X or Linux. Each installer includes all of the software necessary to run out of the box (the stack). The process is simple; just download, click next-next-next and you are done! Bitnami stacks are completely self contained and will not interfere with other software on your system.

Blocking software

turn OFF antivirus and Skype during installation. You can leave default settings asked by Bitnami installer.

To start already installed and configured AbanteCart application click the Go To Application button
