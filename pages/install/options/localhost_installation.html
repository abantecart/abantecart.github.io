Installation on local computer

You do not need to get hosting account to test run or develop with AbanteCart application. If you want to test or develop with AbanteCart, you can install it on your own computer and get convenience of accessing web site and code locally. There are 3 most popular third party applications available that provide virtual layer with web server and database functionality. All are free. Bitnami the simplest way and support few OS, XAMPP is available for MAC, WIN and Linux.

Bitnami Stack

Bitnami AbanteCart Stack provides a one-click install solution for AbanteCart.

XAMPP

XAMPP is a completely free, easy to install Apache distribution containing MySQL, PHP, and Perl.

Denwer

Denwer is more for russian speaking developers and community.

Bitnami AbanteCart Stack

https://bitnami.com/stack/abantecart

Bitnami AbanteCart Stack provides a one-click install solution for AbanteCart. Bitnami native installers automate the setup of a Bitnami AbanteCart application stack on Windows, OS X or Linux. Each installer includes all of the software necessary to run out of the box (the stack). The process is simple; just download, click next-next-next and you are done! Bitnami stacks are completely self contained and will not interfere with other software on your system.

Blocking software

turn OFF antivirus and Skype during installation. You can leave default settings asked by Bitnami installer.

To start already installed and configured AbanteCart application click the Go To Application button

XAMPP

http://www.apachefriends.org/en/xampp.html

Documentation http://www.apachefriends.org/en/faq-xampp.html
Once XAMPP installed and configured:

  • unpack all files and folder from public_html directory in the AbanteCart sources archive to xampp_install_path\xampp\htdocs or to xampp_install_path\xampp\htdocs\abantecart_folder_name\
  • create empty database. Before you must copy the phpMyAdmin folder from the xampp directory and place it into the xampp\htdocs directory. Before testing phpMyAdmin, make sure that both Apache and MySQL are running. Go to http://localhost/phpMyAdmin/ to create new database.
  • start AbanteCart installation: in your browser go to http://localhost/ or http://localhost/abantecart_folder_name/ this should redirect to http://localhost/install/index.php or http://localhost/abantecart_folder_name/install/index.php

when install asked use the following :
Database Host: localhost
User: root
Password: leave empty
Database Name: your new database name

Denwer

http://www.denwer.ru

Denwer http://www.denwer.ru
Documentation http://www.denwer.ru/base.html
Once Denwer installed and configured:

  • create new folder abantecart_folder_name in the \denwer_install_path\home\ directory
  • unpack all files and folder from public_html directory in the AbanteCart sources archive to denwer_install_path\home\abantecart_folder_name\www\ directory.
  • start Denwer (click on Run shortcut on your Desktop) or manually denwer_install_path\denwer\run.exe
  • create empty databasein phpMyadmin. Go to http://localhost/Tools/phpmyadmin/index.php to create new database.
  • start AbanteCart installation: in your browser go to http://abantecart_folder_name/ or http://abantecart_folder_name/install/index.php

when install asked use the following :
Database Host: localhost
User: root
Password: leave empty
Database Name: your new database name
