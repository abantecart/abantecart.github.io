Cpanel Installation

A lot of hosting providers use cPanel. cPanel is hosting control panel. Using Cpanel you able to install AbanteCart in few clicks.

  • 1

    Login to your hosting Cpanel and find Softaculous section.

    In Softaculous Apps Installer click on AbanteCart icon. Manually you can find AbanteCart in Softaculous E-Commerce menu.
  • 2

    Click Install

    In AbanteCart details page click on Install button.
  • 3

    Spot Healing Brush Tool

    Next step is fill all required fields and submit.
  • 4

    Magic Wand Tool

    Thats all. Softaculous automatically create MySql database, Mysql user, copy AbanteCart files and configure shopping cart!
