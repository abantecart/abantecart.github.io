Understanding Locations

What is a locations

Location is a description of some area in the world. It can describe group of countries, country or part of single country. There are number of tasks when you need to use it:

  • setting up some shipping methods based on the customer location
  • setting up tax rates based on the customer location
  • limit payment method to available for customer location
  • limit shipping method to available for customer location

Note

When creating tax rates, make sure you are adhering to the tax laws of your country and locality.

How to create a Location

System > Localization > Locations

Go to System > Localization > Locations section in your admin area.
Click Add button there.
Specify the Location name (so you could easily refer to it) and what Countries and Zones (States) define this Location.
