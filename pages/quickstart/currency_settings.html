List of Currencies

System > Localization > Currencies

FTP software required

Upload all the files to your server by FTP software, e. g. FileZilla.

Add new Currencies

To add a new Currency click the + button in the Currencies list System > Localization > Currencies

Decimal Point

To modify point format go to Admin->System->Localization->Language Definitions and filter grid by decimal_point or thousand_point key.
