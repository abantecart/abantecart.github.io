Methods to install extension

There are several methods to install extension to AbanteCart:

Install via marketplace If available
Install with key If available
Install via package upload
Manual install with FTP or directory copy option

Methods are very easy and do not require special skills.

This document describes the steps you need to perform in order to install an extension or template.

Extensions are available in Admin > Extensions > Get Extensions section or thru browser direct link http://marketplace.abantecart.com

Info

Extensions might be free or paid

  • 1

    In admin go to the Extensions > Get Extensions

  • 2

    Connect your store to the AbanteCart marketplace and Purchase extension. Even if it is free and have 0.00 price.

  • 3

    After purchase you get link and the key to install extension. In case your store connected to the marketplace the installation process run automatically.

Info

You can find Key or Download extension after purchase in your Marketplace account Downloads section.

Install via marketplace If available

After Successful purchase connect your site to AbanteCart Marketplace account and visit My Extensions section.

Extension's compatible with your AbanteCart version can be installed with one click

Install with key If available

Go to Admin > Extensions > Install Extension section. Enter your download key and click Continue button.

Info

Extension key - this is a key to install new extension or update package into AbanteCart. Key can be given after purchase in your order history or will be available for free extensions.

 

Wait while package downloading

After package download, please read the license agreement. If you accept its statements click on the " Agree" button.

 

Congratulations. Your Extension is installed successfully.

Configure Installed Extension. You need to set the settings and enable extension.
Extension might have required or optional extensions in the package or separate.

Install via package upload

Go to Admin > Extensions > Install Extension section. Click on Extension upload Tab.

Info

this method can be used for upgrading to AbanteCart new version. Upgrade packages link can be found at AbanteCart site

Manual install with FTP or directory copy option

  • 1

    download the extension archive.

    Download link in your Marketplace downloads section.
  • 2

    Unarchive it somewhere on your PC.

    A .tar.gz file is a compressed archive similar to zip and RAR files. To extract .tar.gz files on Windows use 7-zip (freeware) or Winzip (commercial). After 7-Zip installed, try to open your .tar.gz file. 7-Zip will start up and decompress the archive. And if it still doesn’t extract, do it manually via the context menu: right-click on the archive and choose one of the “Extract” options.
    Mac OS and Linux OS operating systems can handle .tar.gz file by default. In OS X Just double-click the archive and it will extract to a folder of the same name.
  • 3

    Using any FTP software upload all files.

    Using any FTP software upload all files and folders from the '../extension_name/code/extensions/' directory to 'extensions' directory in your AbanteCart website root folder.
  • 4

    After upload go to Admin > Extensions > All Extensions section .

    Find your extension and click install button.

  • ..

    Congratulations. Your Extension is installed successfully.
