Parcelforce 48

Extensions > Shipping > Parcelforce 48

This extension contains the Parcelforce 48 prices and services, as they do not provide an API.

Fill in the following fields:

Store: - Select your store.

Status: - MUST be "ON" for Shipping module to appear in your store.

Parcelforce 48 Rates: - Enter values up to 5,2 decimal places. (12345.67) Example: .1:1,.25:1.27 - Weights less than or equal to 0.1Kg would cost 1.00, Weights less than or equal to 0.25g but more than 0.1Kg will cost 1.27. Do not enter KG or symbols.

Display Delivery Weight: - Do you want to display the shipping weight? (e.g. Delivery Weight : 2.7674 Kg's)

Display Insurance: - Do you want to display the shipping insurance? (e.g. Insured up to 500).

Display Delivery Time: - Do you want to display the shipping time? (e.g. Ships within 3 to 5 days).

Parcelforce48 Compensation Rates: - Enter values up to 5,2 decimal places. (12345.67) Example: 34:0,100:1,250:2.25 - Insurance cover for cart values up to 34 would cost 0.00 extra, those values more than 100 and up to 250 will cost 2.25 extra. Do not enter currency symbols.

Tax Class: - Select Tax Class.

Location: - select proper location

Storefront Icon: - Shipping Icon to be displayed at checkout shipping selection

Auto Select: - You can set to skip shipping selection page if there is only one shipping option. This shipping method will be selected automatically

Accept these payments: - If only specific payment methods are needed for this shipping method, select them here. If nothing selected, all available payment methods will be shown

Sort Order: - Sort Order set's the order of which the Shipping will be displayed amongst others.

After you have made your changes, select the SAVE button.

Parcelforce 48 not showing up

Double check your product have weight and weight class, length, width, height and dimensions class
