Royal Mail

Extensions > Shipping > Royal Mail

Royal Mail is a premium shipping method that lets you calculate shipping using their rates. This extension contains the Royal Mail prices and services, as they do not provide an API.

Fill in the following fields:

Store: - Select your store.

First Class Standard Post: - First Class mail aims to deliver your letter or packet the next working day, including Saturday..

First Class Recorded Post: - First Class mail aims to deliver your letter or packet the next working day, including Saturday..

Second Class Standard Post: - Second Class mail aims to deliver your letter or packet by the third working day after posting, including Saturday..

Second Class Recorded Post: - Second Class mail aims to deliver your letter or packet by the third working day after posting, including Saturday..

Standard Parcels: - Save money on mailing heavy, non-urgent items across the UK.

Airmail: - Airmail is a fast and cost-effective way to send items worldwide.

International Signed: - Extra reassurance that your item has been delivered to its overseas destination.

Airsure: - Airsure® - a fast, secure and reliable service to over 30 destinations worldwide.

Surface: -Surface Mail – the easy, most affordable way to send items worldwide.

Display Delivery Weight: - Do you want to display the shipping weight? (e.g. Delivery Weight : 2.7674 Kg's)

Display Insurance: - Do you want to display the shipping insurance? (e.g. Insured up to 500).

Display Delivery Time: - Do you want to display the shipping time? (e.g. Ships within 3 to 5 days).

Weight Class: - Set to kilograms or pounds.

Tax Class: - Select Tax Class.

Location: - select proper location

Storefront Icon: - Shipping Icon to be displayed at checkout shipping selection

Auto Select: - You can set to skip shipping selection page if there is only one shipping option. This shipping method will be selected automatically

Accept these payments: - If only specific payment methods are needed for this shipping method, select them here. If nothing selected, all available payment methods will be shown

Status: - MUST be "ON" for Shipping module to appear in your store.

Sort Order: - Sort Order set's the order of which the Shipping will be displayed amongst others.

After you have made your changes, select the SAVE button.

Royal Mail not showing up

Double check your product have weight and weight class, length, width, height and dimensions class
