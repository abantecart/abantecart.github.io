FEDEX

Extensions > Shipping > Fedex

There are three stages to the FedEx setup for AbanteCart:
Register for and set up your basic FedEx account.
Obtain your FedEx credentials.
Activate your FedEx service from your AbanteCart admin.

Fill in the following fields:

Store: - Select your store.

Status: - MUST be "ON" for Shipping module to appear in your store.

FedEx Production Key: - Enter the FedEx Key Assigned to You, Required. You must create an account with FedEx, and this value will provided by them.

FedEx Production Password: - Enter the FedEx Password, required.

Account Number: - Enter the FedEx Account Number assigned to you, required.

FedEx Meter Number: - Enter FedEx Meter Number.

Test Mode: - Use the live or testing gateway server?

Address: - Shipping from street address.

City: - Shipping from city.

State: - Shipping from state (two letters). Example: for New York enter NY

Zip/postal code: - Shipping from Zip-code.

Country: - Shipping from country(two letters). For United States of America enter US

Quote Type: - Quote for Residential or Commercial Delivery.

Additional Charges: - Additional charge applied to FedEx quote.

FEDEX First Overnight: - Select checkbox if You want to use this service.

FEDEX Priority Overnight: - Select checkbox if You want to use this service.

FEDEX Standard Overnight: - Select checkbox if You want to use this service.

FEDEX 2nd Day: - Select checkbox if You want to use this service.

FEDEX Express Saver: - Select checkbox if You want to use this service.

FEDEX Ground: - Select checkbox if You want to use this service.

Storefront Icon: - Shipping Icon to be displayed at checkout shipping selection

Auto Select: - You can set to skip shipping selection page if there is only one shipping option. This shipping method will be selected automatically

Accept these payments: - If only specific payment methods are needed for this shipping method, select them here. If nothing selected, all available payment methods will be shown

Sort Order: - Sort Order set's the order of which the Shipping will be displayed amongst others.

After you have made your changes, select the SAVE button.

Before upgrade

Recommended that you DO NOT upgrade AbanteCart on live site. Depending on your server or OS configuration upgrades might not go all the way through and you will need more time for manual upgrade. We suggest you upgrade your test site first and always create backups.
Upgrades are available only from prior version to next. If you do not upgrade for long time, you will need to run all upgrades in between.
If you are made Core modifications in any of the files that changed, they may need to be redone.
