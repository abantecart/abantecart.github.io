Extensions > Payment > Realex

Realex Payments is one of Europeâ€™s largest and fastest growing online payment gateways, providing a range of payment processing services for businesses selling online.

Fill in the following fields:

Store: - Select your store.

Status: - MUST be "ON" for Payment module to appear in your store.

Merchant ID: - This is the Merchant ID provided by Realex.

Secret: - Your secret password agreed with Realex used for encryption and decryption of message data for security of all transactions.

Rebate password: - Your refund password agreed with Realex used for encryption and decryption of message data for security of refund transactions.

Settlement type: - If set to 'auto', then the transaction will be included in today's settlement. If set to 'delayed', then the transaction will be authorised but not settled. You must manually settle delayed transactions within 28 days of authorization. 'Multi' multiple settle requests can be made to settle varying amounts of the authorization

TSS checks: Enable TSS Realscore service.

Enable 3D secure: - Enables RealMPI Payer Authentication (3D Secure, SecureCode) for Visa, MasterCard and Switch cards only. Note that you must contact Realex support to enable on your account.

Accept 3D failures: Accept non-liability shifting scenarios. You will still accept payments if user fails 3D secure.

Accepted Credit Card Types: - Check with Realex what credit cards your can accept for your country and select from the list.

Order Status: - Select the Order Status of all new Orders created using this payment method from the drop-down. The recommended status is Pending, as this is intended to apply to Orders which have not had any actions performed on them, however you can change this to whichever status you prefer.

Location: - select proper location

Storefront Icon: - Payment Icon to be displayed at checkout payment selection

Total Order Minimum: - Minimum order amount to be reached to allow this payment

Total Order Maximum: - Maximum order amount to be reached to disallow this payment

Auto Select: - You can set to skip payment selection page if there is only one payment option. This payment method will be selected automatically

Sort Order: - Sort Order set's the order of which the Payment will be displayed amongst others.

After you have made your changes, click the SAVE
