Skrill

Extensions > Payment > Skrill

Skrill (Moneybookers) is an all in one payment solution that enables merchants to accept debit and credit card payments, bank transfers and a large range of local payments directly on their website.

Fill in the following fields:

Store: - Select your store.

Status: - MUST be "ON" for Payment module to appear in your store.

E-mail: - Enter your Moneybookers e-mail address.

Order Status: - Select the Order Status of all new Orders created using this payment method from the drop-down.

Order Status Pending: - Select the Order Status of all new Pending Orders created using this payment method from the drop-down. The recommended status is Pending, however you can change this to whichever status you prefer.

Order Status Canceled: - Select the Order Status of all new Canceled Orders created using this payment method from the drop-down. The recommended status is Canceled, however you can change this to whichever status you prefer.

Order Status Failed: - Select the Order Status of all new Failed Orders created using this payment method from the drop-down. The recommended status is Failed, however you can change this to whichever status you prefer.

Order Status Chargeback: - Select the Order Status of all new Chargeback Orders created using this payment method from the drop-down. The recommended status is Chargeback, however you can change this to whichever status you prefer.

Location: - select proper location

Storefront Icon: - Payment Icon to be displayed at checkout payment selection

Total Order Minimum: - Minimum order amount to be reached to allow this payment

Total Order Maximum: - Maximum order amount to be reached to disallow this payment

Auto Select: - You can set to skip payment selection page if there is only one payment option. This payment method will be selected automatically

Sort Order: - Sort Order set's the order of which the Payment will be displayed amongst others.

After you have made your changes, click the SAVE
