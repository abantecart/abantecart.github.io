List General Settings

This page can be accessed by going to System > Settings > General

Default Items per Page (Storefront): Sets how many storefront items are shown per page. Used in all Storefront listings (Category Listing, Search results, Specials page)

Default Items per Page (Admin): Sets how many admin items are shown per page in listing grids (products, manufacturers, orders, customers, etc)

Number of Items In Bestseller Block: Determines how many items are shown in bestsellers block (Store front)

Number of Items In Featured Block : Determines how many items are shown in Featured block (Store front). To set product to appear as featured in your store go to product's edit page

Number of Items In Latest Block : Determines how many items are shown in Latest block (Store front)

Number of Items In Special Block : Determines how many items are shown in Special block (Store front). To set product to appear as special in your store go to product's edit page Promotions Tab

Sort Products as default by : Select default products sort method on Store front

Display Stock : Display stock quantity on the product page.

No Stock Auto Disable : Disable your product or option value for view if stock drops to 0. Message notification will be send.

Stock Status : Default setting for out of stock status message for product. Used on product edit page.
Applies for new products only. See also System -> Localization -> Stock Statuses.
read more about Stock processing configurations

Allow Reviews : Enable/Disable new review entry and display of existing reviews. This will hide all existing reviews on StoreFront.

Allow Downloads : Set your store to allow downloads. If customers make orders with that have downloadable products this setting will stop them from being able to access their downloads.

Help Links : If you want to see Help Icons and links to AbanteCart site. This will help you learn about features and how to use them. If you advanced user and do not want to see these help icons, you can disable.

Enable Tree Grid : Enables admin grids to show parent/children types in one grid listing. NOTE: Pagination and multi-actions are disabled for these grids. Recommended value in ON.

Enable Storefront Embed : Disable Embed feature.

Embed Click Action : Change the way embed store open.

Captcha on create account form : Prevent spam/robot account creation with enabling human verification captcha.

Google reCAPTCHA Site Key : Add Google reCAPTCHA v2 to help your visitors with better captcha entry. If you enable reCAPTCHA site/secret keys, Google reCAPTCHA v2 will be activated instead of default AbanteCart captcha image.

Google reCAPTCHA Secret Key : Add Google reCAPTCHA v2 to help your visitors with better captcha entry. If you enable reCAPTCHA site/secret keys, Google reCAPTCHA v2 will be activated instead of default AbanteCart captcha image. If both keys are emty then default buit-in captcha image used.

Note

reCAPTCHA is a free service that protects your website from spam and abuse while letting real people pass through with ease.

Google Analytics Account Number : Google Analytics is powerful, easy-to-use reporting platform lets you measure sales and conversions, but also gives you fresh insights into how visitors use your site, how they arrived on your site and how you can keep them coming back. Easy connect GA - add your Google Analytics Account Number to enable site activity tracking. This also enables ecommerce activity tracking. You must enable ecommerce tracking in Google on the profile settings page for your website.

After you have made your changes, select the save button
